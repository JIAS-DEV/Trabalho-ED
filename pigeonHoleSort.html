Pigeonhole Sort

A classificação Pigeonhole é um algoritmo de classificação adequado para classificar listas de elementos em que o número n de elementos e o comprimento N do intervalo de valores de chave possíveis são aproximadamente os mesmos. Requer tempo O ( n + N ). É semelhante à classificação por contagem , mas difere por "mover os itens duas vezes: uma vez para a matriz de baldes e novamente para o destino final [enquanto] a classificação por contagem cria uma matriz auxiliar e usa a matriz para calcular o destino final de cada item e mover o artigo lá." 

        
            // PIGEONHOLE SORT
            public long pigeonHoleSort(int vetor[]) { 
                tempoInicial = System.nanoTime();

                int min = vetor[0];
                int max = vetor[0];
                int intervalo, index;

                for(int i = 0; i < vetor.length; i++) {
                        if(vetor[i] > max)
                                max = vetor[i];
                        if(vetor[i] < min)
                                min = vetor[i];
                }
                intervalo = max - min + 1;

                int[] phole = new int[intervalo];
                Arrays.fill(phole, 0);

                for(int i = 0; i < vetor.length; i++)
                    phole[vetor[i] - min]++;
                index = 0;
                for(int i = 0; i < intervalo; i++)
                    while(phole[i] --> 0)
                        vetor[index++] = i + min;
            }
